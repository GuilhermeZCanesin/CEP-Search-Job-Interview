# CepSearch

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.2.9.

## Setup and Running

### 1. Install dependencies

```bash
npm install
```

### 2. Run Back End

```bash
node ./cep-api/server.js
```


### 2. Run Front End

At a new terminal window run:

```bash
npm start
```
